
# 原神抽卡模拟器

## 项目简介
这是一个基于 HarmonyOS 开发的原神抽卡模拟器，实现了完整的抽卡概率和保底机制，能够对抽卡进行近似模拟。原神是一款由上海米哈游公司制作并发行的3D开放世界游戏。
在该游戏中，玩家需要通过抽卡来获得指定角色。

## 功能特点
- ✨ 实现单抽 / 十连抽
- 🎯 理论每抽精确概率：五星 0.6%，四星 6.0%，三星 93.4%，但不包括保底。
- 🔒 双重保底：每10抽必出四星，每90抽必出五星
- 💫 抽卡动画：星星垂直掉落特效；如果是十连抽，将会播放真实的抽卡动画。
- 🎨 稀有度颜色区分：金色/紫色/蓝色
- 📜 抽卡历史记录（最近10次），将会给出四星平均出率和五星平均出率。
- 🖼️ 抽出五星角色将出现立绘弹窗

## 卡池角色
**五星角色**：刻晴、迪卢克、七七、莫娜、琴、甘雨、胡桃

**四星角色**：班尼特、诺艾尔、菲谢尔、砂糖、行秋、重云、香菱

**三星物品**：飞天御剑

## 运行环境
- DevEco Studio 5.0+
- HarmonyOS SDK API 14+

## 如何运行
1. 用 DevEco Studio 打开项目根目录
2. 等待依赖下载完成
3. 启动模拟器（或连接真机）
4. 点击运行 ▶️

修改：已在module.json5中配置了phone,tablet和2in1，理论上支持多端部署,可在手机、平板等设备上运行。
